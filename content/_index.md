# Raccoon Robotics Platform

Welcome to the Raccoon documentation — your complete guide to building and programming competitive Botball robots with
the Raccoon platform.

The Raccoon platform consists of several tools that work together:

- **raccoon-cli** — The command-line toolchain you use on your laptop to create projects, configure hardware, calibrate
  motors, and run your code on the robot.
- **LibStp** — The Python robotics library your mission code uses to control motors, servos, sensors, and plan
  autonomous sequences.
- **Web IDE** — A browser-based editor for visually building mission flowcharts.
- **Robot UI** — The touchscreen interface on the robot itself for monitoring sensors, running programs, and managing
  settings.
