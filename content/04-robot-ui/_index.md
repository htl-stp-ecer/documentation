# Robot UI

The Robot UI is the touchscreen interface displayed on the robot's screen. It gives you direct access to sensor data, running programs, WiFi management, and system settings — without needing a laptop.

- [Settings]({{< ref "/04-robot-ui/03-settings" >}}) — screen rotation, touch calibration, service status
