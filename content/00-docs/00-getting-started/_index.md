# Getting Started

Welcome! This guide will help you get up and running in minutes.

## Prerequisites

Before you begin, make sure you have:

- A modern web browser (Chrome, Firefox, Safari, or Edge)
- Basic understanding of command-line interfaces
- An active account (sign up at example.com)

## Installation

### Step 1: Download

Download the latest version from our [releases page](https://example.com/releases).

```bash
curl -LO https://example.com/download/latest
```

### Step 2: Install

Extract and install the package:

```bash
tar -xzf latest
sudo mv binary /usr/local/bin/
```

### Step 3: Verify Installation

Check that everything is installed correctly:

```bash
binary --version
```

You should see output like: `v1.0.0`

## Quick Start

Create your first project:

```bash
binary init my-project
cd my-project
binary start
```

Visit `http://localhost:8080` in your browser to see your project running!
