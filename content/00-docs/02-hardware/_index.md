# Hardware Library

A collection of proven mechanical designs for Botball robots. Each design includes advantages, disadvantages, build notes, and real competition applications.

The designs are organized into three categories:

**Grabbers**
- [Gravity Grabber](/00-docs/02-hardware/02-gravity-grabber/) -- Spring-loaded, servo-less jaws
- [Horizontal Gear-Driven Grabber](/00-docs/02-hardware/03-horizontal-gear-driven-grabber/) -- Gear-train-driven symmetric jaws
- [Vertical Claw Excavator](/00-docs/02-hardware/04-vertical-claw-excavator/) -- Top-down scooping claw
- [Whip-Spider Pedipalp Arm](/00-docs/02-hardware/08-whip-spider-pedipalp-arm/) -- Bio-inspired barbed arm

**Arms**
- [Simple Rotating Arm](/00-docs/02-hardware/00-simple-rotating-arm/) -- Single beam on a pivot
- [Two-Part Rotating Arm](/00-docs/02-hardware/01-two-part-rotating-arm/) -- Articulated arm with elbow joint

**Lifts & Linear Motion**
- [Pulley Lift](/00-docs/02-hardware/05-pulley-lift/) -- Rail-mounted cart with winch
- [Rack-and-Pinion Slider](/00-docs/02-hardware/06-rack-and-pinion-slider/) -- Gear-driven linear slider
- [Shifting Parallelogram](/00-docs/02-hardware/07-shifting-parallelogram/) -- Four-bar linkage lift
